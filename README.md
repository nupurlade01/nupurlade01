Attribute	Details
Stack	Python · Flask · OpenAI Whisper · HuggingFace Transformers · BERT · BART · NLP
Scale	Multilingual video input · End-to-end inference pipeline
Performance	Real-time audio transcription · Automated AI-generated analysis
Security	Server-side video processing · No third-party data exposure
Impact	Presented at ETLTC-ICETM 2026 International Conference, Japan
Repository	github.com/nupurlade/tatvai
Implemented speech-to-text transcription using OpenAI Whisper across multilingual audio streams

Designed a responsive frontend for real-time output rendering of AI-generated insights

<details>
<summary><strong>◦ SkillXchange — Relational Database Backend System</strong></summary>
<br/>
A fully normalised, constraint-driven MySQL database system engineered to power a skills exchange platform. The schema supports complex multi-entity relationships — users, skills, project requests, messages, and notifications — with advanced SQL objects enabling efficient backend operations and real-time analytics.
Attribute	Details
Stack	MySQL · SQL · DBMS · Relational Schema Design
Scale	Multi-entity relational model · Complex join operations
Performance	Analytical views · Simulated materialized views for reporting
Security	Primary/foreign key constraints · Referential integrity enforced
Impact	Production-ready backend database layer for a full-stack platform
Repository	github.com/nupurlade/skillxchange
Designed a 3NF-compliant normalised schema with primary keys, foreign keys, and cascade constraints
Implemented joins, subqueries, aggregate functions, views, triggers, and stored procedures for backend logic
Built analytical views and a simulated materialized view for efficient user activity summarisation
Ensured data integrity, consistency, and reliability across all transactional operations
</details>
